# Graph Demo Conversation Script Pack

Use these scripts as spoken recordings (upload one by one in order). This is tuned for the current graph logic in `SummarySuggestionsScreen`.

## How To Use

1. Record/upload **Script 1** to **Script 6** in the given order.
2. Keep the same language style (Hindi + English is fine).
3. After Script 6 finishes processing, open the Summary screen.

This ensures:
- **Line Graph** gets multiple recent numeric amounts.
- **Pie Chart** gets clear keyword signals for Loans/SIP/Expenses/Savings.
- **Bar Chart** gets loan context, so comparison options with rates are shown.

---

## Script 1 (Amount baseline)

Speaker A: I am planning a home loan of **8 lakh** for a small apartment.

Speaker B: Okay, 8 lakh noted. We should estimate EMI and eligibility.

Speaker A: My expected **EMI** should stay around **18,000 to 22,000 rupees** per month.

## Why This Works

- **Line Graph**: Scripts 1-6 include explicit numeric amounts with units (lakh), so recent timeline has data points.
- **Pie Chart**: Script 6 includes category keywords: loan, SIP, expenses, savings.
- **Bar Chart**: Script 6 is clearly loan-focused, so the app generates comparison options with percentage rates.

---

## Quick Validation Checklist

After Script 6:
- Summary shows loan topic and a numeric amount.
- Line graph shows at least 2 points.
- Pie chart shows non-empty percentage segments.
- Bar chart shows at least 2 providers with rate percentages.
